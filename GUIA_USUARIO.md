# 📋 Guia do Usuário - Sistema de Transferência de Material

## 🚀 Como Usar o Sistema

### 1️⃣ **Acessando o Sistema**
- Abra seu navegador
- Digite: `https://192.168.2.96:5173`
- ⚠️ Se aparecer aviso de segurança, clique em "Avançado" → "Prosseguir para 192.168.2.96"

### 2️⃣ **Consultando um Material**

1. **Digite o código da etiqueta** no campo de busca
   - Exemplo: `AA162042`
   - ✅ Use apenas o código da etiqueta (sem espaços)

2. **Clique no botão "Consultar"**
   - O sistema irá buscar os dados do material
   - Aguarde alguns segundos para o carregamento

3. **Visualize as informações**
   - ✅ **Código do Material**: Código interno do sistema
   - ✅ **Status**: Se há requisição ativa
   - ✅ **Solicitante**: Quem solicitou o material
   - ✅ **Tipo**: Categoria do material
   - ✅ **Localização**: Onde está atualmente
   - ✅ **Depósito Origem**: De onde sairá
   - ✅ **Depósito Destino**: Para onde irá

### 3️⃣ **Transferindo o Material**

1. **Após consultar o material**
   - O botão "Transferir" ficará habilitado automaticamente
   - ⚠️ Só aparece se houver requisição válida

2. **Clique em "Transferir"**
   - O sistema executará a transferência
   - Aguarde a confirmação

3. **Confirmação**
   - ✅ **Sucesso**: Mensagem verde + som de confirmação
   - ❌ **Erro**: Mensagem vermelha com detalhes

## 🔍 **Exemplos Práticos**

### ✅ **Exemplo de Uso Correto**
```
1. Digite: AA162042
2. Clique: "Consultar"
3. Veja os dados carregados
4. Clique: "Transferir"
5. Aguarde confirmação
```

### ❌ **Erros Comuns**

| Problema | Causa | Solução |
|----------|-------|----------|
| "Material não encontrado" | Código incorreto | Verifique o código da etiqueta |
| Botão "Transferir" desabilitado | Sem requisição ativa | Material não tem requisição válida |
| "Erro na transferência" | Problema no sistema | Contate o suporte técnico |
| Página não carrega | Servidor offline | Verifique se o sistema está rodando |

## 🎯 **Dicas Importantes**

### ✅ **Boas Práticas**
- Sempre consulte antes de transferir
- Verifique os dados antes de confirmar
- Aguarde a confirmação completa
- Use códigos de etiqueta corretos

### ⚠️ **Cuidados**
- Não feche a página durante a transferência
- Não clique múltiplas vezes no botão
- Verifique se o depósito destino está correto
- Em caso de erro, anote a mensagem para suporte

## 🆘 **Resolução de Problemas**

### **Problema: Página não carrega**
```
1. Verifique se o servidor está rodando
2. Tente atualizar a página (F5)
3. Limpe o cache do navegador
4. Tente outro navegador
```

### **Problema: Erro de certificado**
```
1. Clique em "Avançado"
2. Clique em "Prosseguir para 192.168.2.96"
3. Isso é normal em ambiente local
```

### **Problema: Material não encontrado**
```
1. Verifique se o código está correto
2. Confirme se é o código da etiqueta
3. Tente sem espaços ou caracteres especiais
```

## 📞 **Suporte**

Em caso de problemas:
1. Anote a mensagem de erro exata
2. Informe o código do material usado
3. Descreva o que estava fazendo
4. Contate o suporte técnico
