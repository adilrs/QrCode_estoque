📦 Manual do Usuário - Sistema de Transferência de Materiais

Sistema Materiais Versão 1.0 Status Ativo

🎯 Visão Geral

🚀 Sistema inteligente para consulta e transferência de materiais entre depósitos, com leitura de QR codes e validação automática de saldos.

📋 Como Usar

🔍 1. Consulta de Material

📱 Opção A: Escaneamento de QR Code

  1. 🎯 Clique no botão "Escanear QR Code"
  2. 📷 Aponte a câmera para o código QR do material
  3. ⚡ O sistema carregará automaticamente as informações

⌨️ Opção B: Busca Manual

  1. 📝 Digite o código do material no campo de busca
  2. 🔍 Clique em "Buscar"
  3. 📊 As informações serão exibidas na tela

📊 2. Informações Exibidas

💡 Após a consulta, você verá um painel completo com:
🏷️ Campo 📝 Descrição
🔢 Código Identificação única do material
📄 Descrição Nome/descrição detalhada do material
📍 Localização Onde o material está fisicamente localizado
🏷️ Qtd. Etiqueta Quantidade registrada na etiqueta
📦 Qtd. Requerida Quantidade solicitada na requisição
📋 Requisição Nº Número da requisição (quando aplicável)
⚡ Status Situação atual do material no sistema
🏪 Dep. Origem Depósito de origem com saldo disponível
🎯 Dep. Destino Depósito de destino da transferência

🚚 3. Transferência de Material

✅ Quando Transferir

🟢 O botão "Transferir" estará disponível quando:
  • ✅ Houver uma requisição válida
  • ✅ O saldo do depósito de origem for suficiente
  • ✅ O material não tiver sido transferido anteriormente

⚠️ Situações Especiais

🚫 Status 📝 Significado
"Sem Requisição" Material não possui requisição ativa
"Já Transferido" Material já foi transferido anteriormente
"Saldo Insuficiente" Não há quantidade suficiente no depósito

🎯 Como Transferir

  1. 🔍 Verifique se todas as informações estão corretas
  2. 💰 Confirme que o saldo é suficiente
  3. 🖱️ Clique no botão "Transferir"
  4. ⏳ Aguarde a confirmação do sistema
📋 Observação Importante - Tratamento de Etiquetas:
Quando a quantidade da etiqueta for menor que a quantidade requisitada, será efetuada a baixa da etiqueta com a quantidade requisitada, e será gerada uma nova etiqueta com a quantidade restante com uma nova sequência, voltando assim o saldo e possibilitando nova requisição do volume. Esta etiqueta a ser impressa é enviada para o Sistema Retesp 2000 na lista de etiquetas não impressas, que deverá ser impressa e colada no volume retornado.

📋 4. Requisições Pendentes

🎯 Fluxo de Trabalho:

📋 Requisições → 👀 Visualizar → 🖱️ Clicar → 🔍 Consultar → 🚚 Transferir
  1. 📋 Clique no botão "Requisições Pendentes"
  2. 👀 Visualize a lista de materiais aguardando transferência
  3. 🖱️ Clique em qualquer etiqueta para consultar automaticamente
  4. 🚚 Proceda com a transferência conforme necessário

💡 Dicas Importantes

✅ Boas Práticas

  • 🔍 Sempre verifique o saldo antes de transferir
  • 🎯 Confirme o depósito de destino
  • ⏳ Aguarde a confirmação antes de prosseguir
  • 📱 Use o QR code sempre que possível
  • 🔄 Mantenha os dados atualizados

⚠️ Atenção

  • 🚫 Materiais sem requisição não podem ser transferidos
  • 💰 Saldo insuficiente impede a transferência
  • ✅ Materiais já transferidos não podem ser transferidos novamente
  • 📶 Conexão estável é necessária
  • 🔒 Dados sensíveis - mantenha segurança

🔧 Problemas Comuns

📱 QR Code não funciona
  • 📷 Verifique se a câmera está funcionando
  • 💡 Certifique-se de que o código está bem iluminado
  • ⌨️ Use a busca manual como alternativa
  • 🔄 Tente limpar a lente da câmera
🚫 Botão Transferir desabilitado
  • ✅ Verifique se há requisição válida
  • 💰 Confirme se o saldo é suficiente
  • 🔍 Verifique se o material não foi transferido
  • 🔄 Atualize a página se necessário
🌐 Erro de conexão
  • 📶 Verifique sua conexão com a rede
  • ⏳ Tente novamente após alguns segundos
  • 🔄 Recarregue a página
  • 📞 Entre em contato com o suporte se persistir
